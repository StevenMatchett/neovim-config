## Neo vim short cuts

CTRL-W w — switch panel

CTRL-O — go back

gd — go to def

q{letter}…q  — record macro

[number]== — indent

Visual mode highlight - = | indent everything highlighted

gg = G — indent all file 

gc — comment toggle

{leader} sg — live grep project

CTRL p — find file

CTRL 6 — previous file

gt — next tab

gT - prev tab

tabnew [filename]

tabc — close tab
